cyperus user manual

welcome to the cyperus realtime music system

this linux software enables a computer to process and generate musically-inclined signals in realtime. it is a client of the Jack Audio Connection Kit (http://jackaudio.org).


please file questions, concerns, and bugs on github or send directly to me at murray@petrichor.systems.

1. required dependencies

  • linux, maybe mac os x
  • alsa and alsa development libraries - (http://alsa-project.org/)
  • JACK server installed and running with development libraries - (http://jackaudio.org)
  • liblo - lightweight OSC implementation (http://liblo.sourceforge.net)
  • FFTW - C subroutines for the discrete Fourier transform (http://www.fftw.org/)

2. building and installation

  1. Clone the cyperus github repository 

    $ git clone https://github.com/petrichorsystems/cyperus

  2. Descend into repository directory and build cyperus. 

    $ cd cyperus
$ make

3. runtime options


$ ./cyperus -h
welcome to the cyperus realtime music system

Usage: cyperus [options] [arg]

Options:
 -h,  --help          displays this menu
 -i,  --input         input channels.  default: 8
 -o,  --output        output channels. default: 8
 -b,  --bitdepth      set bitdepth of capture to 8, 16, 24, 32, 64, or 128. default: 24
 -p,  --port          set osc interface receiving port. default: 97211
 -sp, --send-port     set osc interface sending port. default: 97217
 -f,  --file          set path of session file to load preexisting sounds.
 -fi, --fifo-size     set fifo buffer size for each channel. default: 2048

5. getting started

  1. Start jackd and then cyperus or start cyperus and jackd will be started automatically with your saved jackd settings. 

    $ jackd -d alsa -p 128 &
$ ./cyperus

  2. By default, cyperus will start up a a JACK client and an OpenSoundControl server with the listed defaults.
